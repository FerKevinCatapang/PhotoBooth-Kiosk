Alternative: Laptop Print Server (silent, automatic)

Your laptop runs the print server, which also uploads photos to Drive silently using a service account — no login popup, perfect for a kiosk. Enable Local Print Server mode in Printer Setup first.

  1. In Google Cloud Console, go to IAM & Admin → Service Accounts → create a service account.
  2. Click the service account → Keys → Add Key → JSON. Download the file.
  3. Copy it to your print-server/ folder and name it service-account.json.
  4. In Google Drive, create a folder (e.g. Photo Booth Captures). Right-click → Share → paste the service account email (found in the JSON file under client_email) and give it Editor access.
  5. In print-server/server.js, set the DRIVE_FOLDER_ID variable to the folder's ID (the string after /folders/ in the Drive URL).
  6. Run npm install inside print-server/ to install the googleapis package, then restart the server.

Tablet & Mobile Printing

Running on an iPad or Android tablet? Choose how photos are sent to the printer.

The IP address of the machine running server.js on your local network.

Quick start: On a Raspberry Pi or any machine on the same WiFi, run: 
cd print-server && npm install && node server.js
Then set the URL above to http://<pi-ip>:3000. Set PRINTER_NAME env variable to target a specific printer (leave blank for system default).
iPad setup guide
  1. Connect the Canon SELPHY CP1500 to the same WiFi as the iPad.
  2. On the SELPHY, enable Wi-Fi (menu → Wi-Fi settings → Infrastructure mode).
  3. The printer will appear automatically in iOS AirPrint — no driver install needed.
  4. Use System Print Dialog mode above. When a guest takes a photo, an AirPrint sheet appears briefly and prints silently.
  5. For fully silent printing (no sheet), switch to Local Print Server mode with a Raspberry Pi.
Android tablet setup guide
  1. Install Canon PRINT Inkjet/SELPHY from the Google Play Store.
  2. Open the Canon app once and add the SELPHY printer — this registers it with Android's print subsystem.
  3. Connect the SELPHY to the same WiFi as the tablet.
  4. Use System Print Dialog mode. When a guest takes a photo, the Android print sheet appears with the SELPHY listed.
  5. For a completely dialog-free experience, use Local Print Server mode.

📹 Live Gallery Viewer

Let a second device — a TV, tablet, or phone — display every capture in real-time over your local Wi-Fi network.

How it works

1. Click Start Hosting below. A unique QR code appears.

2. On the viewer device, open the camera app and scan the QR code — it opens a full-screen gallery page.

3. Every new photo or video captured on this device appears on the viewer instantly over your local Wi-Fi.

Note: An internet connection is needed for the first few seconds to establish the peer-to-peer link. After that, all data flows directly over your local network.

Network Address

The QR code only works on other devices when it points to this device's local network IP, not localhost. Enter the address mobile devices can reach this page at.

Find it in: Wi-Fi settings → your connected network → IP address. Then enter http://<IP> here.
